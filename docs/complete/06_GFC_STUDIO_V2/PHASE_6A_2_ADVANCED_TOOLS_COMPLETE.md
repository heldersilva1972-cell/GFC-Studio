# PHASE 6A-2: ADVANCED BUILDER TOOLS (UI/UX)

**Objective:** Enhance the Visual Editor with professional-grade tools for animations, layouts, and reusability.

## 1. Animation Orchestrator
- [ ] **Timeline Component**: A visual bar at the bottom of the screen.
- [ ] **Sequencing Logic**: Interface to set "Delay" and "Duration" graphically.
- [ ] **Preview Mode**: A "Play" button that runs the CSS animations in the canvas without saving.

## 2. Template Library
- [ ] **Template Model**: `StudioTemplates` table (Name, Category, ContentJson).
- [ ] **Save as Template**: Context menu action on any Section -> "Save to Library".
- [ ] **Library Panel**: A new tab in the "Left Toolbox" to drag-and-drop saved templates.

## 3. The "Wizard" AI
- [ ] **Layout Doctor**: Simple logic to check if a "Row" has too many columns for mobile view.
- [ ] **SEO Health Bar**: Real-time checker for H1 tags, Alt text, and meta description length.
