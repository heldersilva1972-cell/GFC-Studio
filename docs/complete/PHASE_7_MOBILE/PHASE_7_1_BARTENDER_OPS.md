# PHASE 7-1: BARTENDER EXPERIENCE (Operational)

**Objective:** Create the mobile-first tools for daily bar operations.

## 1. Mobile Authentication
- [ ] **PIN Login**: Implement a simplified 4-digit PIN login for quick staff access on shared tablets.
- [ ] **Shift Dashboard**: The "Home Screen" for a bartender logic (shows active shift status).

## 2. Shift Management
- [ ] **Clock In/Out**: Logic to record timestamp `Start` and `End`.
- [ ] **Shift Report Wizard**: A step-by-step mobile form to close the night:
    1.  **Count Drawer**: Enter cash total.
    2.  **Bar Sales**: Enter Z-Tape Total.
    3.  **Lotto**: Enter Lotto Sales / Payouts.
    4.  **Notes**: Any issues (broken glass, etc.).

## 3. Inventory Checks (Lite)
- [ ] **Keg Tracker**: Simple buttons to mark a keg as "Kicked" or "New Tapped".
