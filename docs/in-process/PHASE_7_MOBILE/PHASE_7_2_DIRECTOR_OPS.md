# PHASE 7-2: DIRECTOR EXPERIENCE (Management)

**Objective:** Create the "Pocket Command" tools for Directors to manage the club remotely.

## 1. Remote Access Control
- [ ] **Door Command**: Interface to view current Door Status (Locked/Unlocked) and toggle it.
- [ ] **Security**: Implementation of 2FA or higher privilege check for this specific action.

## 2. Member Lookup
- [ ] **Quick Search**: A fast, mobile-optimized list to look up a member by Name or Number.
- [ ] **Status View**: Visual indicator (Green/Red) to show if they are Active/Paid for the current year.

## 3. Alerts & Approvals
- [ ] **Push Notifications**: Integration (PWA or Native) to receive alerts.
- [ ] **Approval Action**: Ability to "Approve" a Hall Rental request directly from the notification.
